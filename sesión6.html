Sesión 6

Hoy tocó la clase virtual así que ingresé a la plataforma y clase, el profe nos contó un poco sobre lo que hariamos en clase de manera resumida.

Dando inicio a la clase, continuamos con el CSS y para empezar hacemos un breve repaso del css y HTML, vimos estructuras de una regla y la configuración de letra y colores. Tambipen vimos como se declara el Id y como se hace la llamada "cascada", el profe además nos mostro como usar el CSS en HTML para poder entender de una mejor manera lo que nos explica, así igual aplicamos color a las letras de nuestro HTML y de igual manera aprendimos a aplicar el background en nuestras páginas.

Nos tomamos un break y regresando de este hicimos un ejercicio de escala de colores, mediante se iba explicando el profesor iba realizando los pasos con nosotros para poder ayudarnos. Me perdí un poquito pero al final lo conseguí y bueno eso fue todo fue un dpia productivo, nos vemos en la siguiente clase! Byeee.
